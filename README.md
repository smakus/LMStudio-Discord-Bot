# LMStudio-Discord-Bot

All credit goes to [DJ Johnson's original post found here](https://dev.to/mrdjohnson/i-made-a-discord-bot-with-lmstudiojs-4fd6).

Follow that basic guide, then use the repository here as a guide for a working version of the node script that lives in the /src directory of your bot folder.   This updated node-based discord bot interfaces with LM Studio so you can run a local LLM for your Discord server.  I made a few tweaks so that it is updated to actually work, plus supports basic chat history.
